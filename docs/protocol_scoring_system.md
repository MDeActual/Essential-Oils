
# Protocol Scoring System — Phyto.ai

Purpose
Allow the platform to automatically evaluate and rank protocols based on real-world contributor outcomes.

Score Components
1. Outcome Improvement Rate
2. Adherence Rate
3. Completion Rate
4. Sample Size Weight

Example Formula

Protocol Score =
(Improvement Rate × 0.4) +
(Adherence Rate × 0.3) +
(Completion Rate × 0.2) +
(Sample Size Weight × 0.1)

Outcome Definitions
Improvement Rate → % of participants reporting improvement
Adherence Rate → average completion of protocol steps
Completion Rate → % of users finishing protocol
Sample Size Weight → confidence factor based on participation

Score Uses
• protocol ranking
• protocol recommendations
• protocol evolution triggers
• archetype-specific performance analysis

Integrity Filters
Exclude:
• adherence < 50%
• incomplete outcomes
• synthetic data
• flagged anomalies

Outputs
Protocol Leaderboard
Best protocols per symptom category
Archetype-specific performance insights
