# DevOS Artifacts

This folder contains scoped DevOS governance artifacts used as reusable execution templates.

These files are not part of the Phyto.ai runtime application code in `src/`.

## Contents

- `prompts/SWARM_PROMPT_STAGED.md` — staged swarm orchestration prompt template
- `runbooks/devos_execution_plan.md` — production-readiness runbook template

These artifacts are retained here for reference and reuse, but should be treated as cross-project DevOS materials rather than Phyto.ai runtime implementation files.
